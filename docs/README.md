## 📂 Rag_Study 문서 구조

| 디렉터리 | 설명 | 주요 내용 |
|----------|------|-----------|
| `guides/` | 아키텍처, 기능, 배포, 업로드 등 실무 가이드 | `ARCHITECTURE.md`, `LANGCHAIN_VS_LANGGRAPH.md`, `MMR_SEARCH_GUIDE.md`, `TEST_EXECUTION_GUIDE.md` |
| `reports/` | 주차별 활동/성과 보고서 | `week1~week7` 리포트, 모델/임베딩 선택 근거 |
| `notes/` | 연구노트 & 구노 업로드용 요약 | `rag_study_note/01~10` |
| `results/` | 실제 실행 결과와 커리큘럼 준수 상태 | `ACTUAL_EXECUTION_RESULTS.md`, `CURRICULUM_COMPLIANCE_CHECK.md`, `IMPLEMENTATION_SUMMARY.md` |

### 참고
- 모든 문서는 루트 `README.md`와 연동되어 있으며, 필요한 경우 GitHub 업로드 가이드를 참고하세요.
- 새 문서를 추가할 때는 적절한 카테고리에 배치하고, 본 README에 항목을 업데이트하면 전체 구조 파악이 쉬워집니다.
